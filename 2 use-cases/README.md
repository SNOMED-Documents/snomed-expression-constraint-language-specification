# 2. Use Cases

The SNOMED CT Expression Constraint Language enables the intensional definition of a bounded set of clinical meanings. This is important for a number of use cases, including:

  * [Terminology Binding](2.1-Terminology-Binding_28739391.html);
  * [Intensional Reference Set Definitions](2.2-Intensional-Reference-Set-Definitions_28739392.html);
  * [SNOMED CT Content Queries](2.3-SNOMED-CT-Content-Queries_28739393.html); and
  * [SNOMED CT Concept Model](2.4-SNOMED-CT-Concept-Model_28739394.html).

In the following subsections, we describe each of these key use cases.
