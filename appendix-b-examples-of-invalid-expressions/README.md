# Appendix B – Examples Of Invalid Expressions

This appendix provides examples of expressions (both precoordinated and postcoordinated) which **do not** satisfy the given expression constraints from [Broken link](broken-reference "mention"). This list of examples is not intended to be exhaustive, but rather to provide a useful sample to help clarify the meaning of these constraint. Please refer to the [SNOMED CT Languages Github repository](https://github.com/IHTSDO/SNOMEDCT-Languages) for a set of text files containing each of these examples.
