# 3. Requirements

In this chapter, we state the requirements of the [SNOMED CT Expression Constraint Language](http://snomed.org/ecl). These requirements are grouped into [General SNOMED CT Language Requirements](../../3%20requirements/3.1-General-SNOMED-CT-Language-Requirements_28739396.html) (which are shared by all SNOMED CT computable languages), [Expression Constraint and Query Requirements](../../3%20requirements/3.2-Expression-Constraint-and-Query-Requirements_28739397.html), and [Concept Model Requirements](../../3%20requirements/3.3-Concept-Model-Requirements_28739398.html).
